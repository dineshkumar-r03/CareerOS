# CareerOS – Student Guidance System

## 🚀 About the Project

CareerOS is a smart student guidance platform that helps students explore career paths, access blogs, and track their learning roadmap.

## ✨ Features

* User Login System
* Dashboard with career insights
* Blog section for learning
* Personalized roadmap

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📌 Future Improvements

* AI-based career suggestions
* Resume analysis
* Job matching system
